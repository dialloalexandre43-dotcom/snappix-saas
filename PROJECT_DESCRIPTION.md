# Snappix - Project Description for Google Chrome Web Store

## Project Name
**Snappix**

## Category
**Productivity** or **Shopping** (Recommended: **Productivity**)

## Short Description (132 characters max)
Generate professional product visuals from Amazon and AliExpress with AI-powered image generation. Transform product photos into marketing-ready visuals.

## Full Description

### Overview
Snappix is a powerful Chrome extension that helps e-commerce sellers, marketers, and content creators generate professional product visuals directly from Amazon and AliExpress product pages. Transform ordinary product photos into stunning, marketing-ready images with AI-powered style generation.

### Key Features

**🎯 One-Click Image Generation**
- Automatically detects product images from Amazon and AliExpress pages
- Instantly generates professional visuals with multiple style options
- No design skills required - just click and generate

**🎨 Multiple Professional Styles**
- **Studio Blanc**: Clean white background for Amazon listings
- **Lifestyle Maison**: Instagram-ready lifestyle scenes
- **Minimal Gradient**: Modern, clean aesthetic
- **Social Ads**: Optimized for Meta and TikTok ads
- **Luxury Black**: High-end, premium look
- **Packshot 3D**: Premium 3D product shots
- **Color Block**: Vibrant, Gen Z-friendly designs
- **Mockup Scene**: Storytelling-focused compositions

**📐 Multiple Format Options**
- Square formats (1024×1024, 1080×1080, 2048×2048)
- Horizontal formats (1920×1080, 16:9 ratio)
- Vertical formats (1080×1920, 9:16 for Stories)
- Custom sizes based on subscription plan

**💼 Subscription Plans**
- **FREE**: 3 images/month, 1 size (1024×1024), 2 styles, watermark
- **STARTER**: 50 images/month, 5 sizes, 8 styles, no watermark
- **PRO**: 300 images/month, 10+ sizes, 15+ styles, no watermark, source exports

**📊 Dashboard Integration**
- View all generated images in your personal dashboard
- Track job status and progress
- Download images individually or in bulk
- Manage your subscription and usage

**🔒 Secure & Private**
- Secure authentication via Google OAuth
- All images stored securely
- Stripe-powered payment processing
- No data sharing with third parties

### Use Cases
- **E-commerce Sellers**: Create professional product images for Amazon, eBay, Shopify listings
- **Social Media Marketers**: Generate Instagram, TikTok, and Facebook ad visuals
- **Content Creators**: Transform product photos into engaging social media content
- **Agencies**: Bulk generate visuals for multiple clients
- **Dropshippers**: Quickly create marketing materials for AliExpress products

### How It Works
1. Install the Snappix Chrome extension
2. Navigate to any Amazon or AliExpress product page
3. Click the Snappix icon in your browser toolbar
4. Select your preferred style and format
5. Click "Generate Now" - your images are created instantly
6. View and download your visuals from your dashboard

### Technical Details
- **Platform**: Chrome Extension (Manifest V3)
- **Backend**: Next.js SaaS application
- **Authentication**: NextAuth.js with Google OAuth
- **Payment**: Stripe integration
- **Image Processing**: AI-powered generation via Fal.ai API
- **Storage**: Secure cloud storage for generated images

### Privacy & Security
- All user data is encrypted and stored securely
- Payment information is processed through Stripe (PCI compliant)
- No product data is stored without user consent
- Images are generated on-demand and stored securely
