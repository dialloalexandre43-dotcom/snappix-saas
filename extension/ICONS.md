# Icônes de l'extension

Les icônes de l'extension sont manquantes. Pour créer les icônes :

## Option 1 : Créer des icônes simples

Vous pouvez créer 3 fichiers PNG :
- `icon16.png` : 16x16 pixels
- `icon48.png` : 48x48 pixels  
- `icon128.png` : 128x128 pixels

Utilisez un outil comme :
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- [Photopea](https://photopea.com) (gratuit, en ligne)

## Option 2 : Utiliser un générateur en ligne

1. Allez sur https://www.favicon-generator.org/
2. Uploadez une image
3. Téléchargez les différentes tailles
4. Renommez-les selon les noms requis

## Option 3 : Utiliser des icônes temporaires

Pour tester rapidement, vous pouvez créer des fichiers PNG simples avec un fond coloré et du texte "SaaS" ou "IMG".

## Note

L'extension fonctionnera même sans icônes, mais Chrome affichera une icône par défaut.
