# Images de Style - Snappix Extension

Ce dossier contient les images de prévisualisation pour les différents styles disponibles dans l'extension.

## Structure des fichiers

Les images doivent être nommées dans l'ordre suivant :

1. `style-1.jpg` - Studio Blanc (dispositif électronique argenté sur fond dégradé rose/violet)
2. `style-2.jpg` - Lifestyle Maison (bouteille beige sur fond gris clair)
3. `style-3.jpg` - Minimal Gradient (bureau avec livres, plantes, lampe)
4. `style-4.jpg` - Social Ads (écouteur blanc AirPods sur fond blanc)
5. `style-5.jpg` - Luxury Black (montre chronographe rose or sur fond noir)
6. `style-6.jpg` - Packshot 3D (bouteille d'eau en acier avec plantes)
7. `style-7.jpg` - Color Block (baskets rouges sur fond géométrique coloré)
8. `style-8.jpg` - Mockup Scene (interface popup avec homme et smartphone)

## Formats supportés

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Dimensions recommandées

- **Taille** : 80x80 pixels minimum
- **Ratio** : 1:1 (carré)
- **Format** : JPG recommandé pour un meilleur poids

## Comment ajouter les images

1. Placez vos images dans ce dossier (`extension/styles/`)
2. Nommez-les selon la structure ci-dessus (`style-1.jpg`, `style-2.jpg`, etc.)
3. Rechargez l'extension dans Chrome (Extensions > Recharger)

Si une image n'est pas trouvée, un gradient de couleur sera affiché à la place.
