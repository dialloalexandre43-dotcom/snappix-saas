# Guide : Ajouter les images de style

## 📋 Ordre des images

Placez vos images dans le dossier `extension/styles/` avec les noms suivants, dans cet ordre :

1. **style-1.jpg** → Studio Blanc
   - Dispositif électronique cylindrique argenté sur fond dégradé rose/violet

2. **style-2.jpg** → Lifestyle Maison
   - Bouteille beige sans marque sur fond gris clair

3. **style-3.jpg** → Minimal Gradient
   - Bureau avec livres, plantes, lampe (ambiance lifestyle)

4. **style-4.jpg** → Social Ads
   - Écouteur blanc (AirPods) sur fond blanc

5. **style-5.jpg** → Luxury Black
   - Montre chronographe rose or sur fond noir

6. **style-6.jpg** → Packshot 3D
   - Bouteille d'eau en acier inoxydable avec plantes

7. **style-7.jpg** → Color Block
   - Baskets rouges sur fond géométrique coloré

8. **style-8.jpg** → Mockup Scene
   - Interface popup avec homme et smartphone

## 📐 Spécifications techniques

- **Format** : JPG, PNG ou WEBP
- **Taille recommandée** : 80x80 pixels minimum (ratio 1:1 carré)
- **Poids** : Optimiser pour le web (< 50KB par image)

## ✅ Étapes

1. Placez vos 8 images dans `extension/styles/`
2. Nommez-les exactement : `style-1.jpg`, `style-2.jpg`, etc.
3. Rechargez l'extension dans Chrome :
   - Ouvrez `chrome://extensions/`
   - Cliquez sur "Recharger" sur l'extension Snappix

## 🔄 Fallback

Si une image n'est pas trouvée ou ne charge pas, un gradient de couleur sera affiché automatiquement à la place.

## 📝 Note

Les images sont maintenant configurées dans le code. Il suffit d'ajouter les fichiers dans le bon ordre !
