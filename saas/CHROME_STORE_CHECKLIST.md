# 📦 Checklist : Publication Chrome Web Store

## ✅ Préparation

### Fichiers requis
- [ ] `manifest.json` (vérifié et à jour)
- [ ] `popup.html`, `popup.js`, `popup.css`
- [ ] `content.js`
- [ ] `background.js`
- [ ] `logo.png` (128x128px minimum)
- [ ] Icônes : `icon-16.png`, `icon-48.png`, `icon-128.png`
- [ ] Images de styles : `styles/*.jpg`

### Configuration
- [ ] URL API par défaut mise à jour avec votre domaine de production
- [ ] Version du manifest mise à jour
- [ ] Description claire et professionnelle
- [ ] Toutes les permissions justifiées

### Assets marketing
- [ ] **Icône de la boutique** : 128x128px (PNG)
- [ ] **Captures d'écran** : 
  - 1 capture minimum, 5 maximum
  - Format : 1280x800px ou 640x400px
  - Format : PNG ou JPEG
- [ ] **Image promotionnelle** : 440x280px (optionnel mais recommandé)
- [ ] **Petite image promotionnelle** : 920x680px (optionnel)

### Contenu
- [ ] **Nom** : Snappix (ou votre nom de marque)
- [ ] **Description courte** : 132 caractères max
- [ ] **Description détaillée** : Explique les fonctionnalités
- [ ] **Catégorie** : Productivité
- [ ] **Langue** : Français (et autres si multilingue)

### Politique de confidentialité
- [ ] Page créée sur votre site : `https://votre-domaine.com/privacy`
- [ ] Contenu conforme aux exigences Chrome
- [ ] Accessible publiquement (pas de login requis)

## 📦 Structure du ZIP

Le fichier ZIP doit contenir :
```
extension.zip
├── manifest.json
├── popup.html
├── popup.js
├── popup.css
├── content.js
├── background.js
├── logo.png
├── icon-16.png
├── icon-48.png
├── icon-128.png
└── styles/
    ├── style1.jpg.jpg
    ├── style2.jpg.jpg
    └── ...
```

**Important** : Les fichiers doivent être à la racine du ZIP, pas dans un dossier `extension/`

## ⚠️ Points d'attention

- **Permissions** : Justifiez chaque permission dans la description
- **Politique de confidentialité** : Obligatoire, doit être accessible
- **Contenu** : Pas de contenu offensant ou illégal
- **Fonctionnalités** : L'extension doit fonctionner comme décrit
- **Tests** : Testez l'extension avant de soumettre
