# Ajouter l'image Before/After

Pour ajouter l'image Before/After à la section du site :

1. Placez votre image dans ce dossier (`saas/public/`)
2. Nommez-la `before-after.jpg` (ou modifiez le nom dans `before-after-section.tsx` si vous utilisez un autre nom)
3. Formats supportés : `.jpg`, `.jpeg`, `.png`, `.webp`

L'image sera automatiquement affichée dans la section "Before / After" de la page d'accueil.

**Emplacement actuel du composant :**
- `saas/components/before-after-section.tsx`

**Chemin de l'image dans le code :**
- `/before-after.jpg` (accessible depuis `public/before-after.jpg`)
