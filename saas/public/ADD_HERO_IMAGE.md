# Ajouter l'image Hero (titre principal)

Pour ajouter l'image en arrière-plan du titre principal :

1. Placez votre image dans ce dossier (`saas/public/`)
2. Nommez-la `hero-background.png` (ou modifiez le nom dans `hero-section.tsx` si vous utilisez un autre nom)
3. Formats supportés : `.png`, `.jpg`, `.jpeg`, `.webp`

L'image sera automatiquement affichée en arrière-plan du titre principal avec une transparence de 20-30%.

**Emplacement actuel du composant :**
- `saas/components/hero-section.tsx`

**Chemin de l'image dans le code :**
- `/hero-background.png` (accessible depuis `public/hero-background.png`)

**Note :** L'image est positionnée en haut à droite et a une opacité de 20% sur mobile et 30% sur desktop pour rester discrète.
